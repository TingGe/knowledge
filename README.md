# 打造团队智慧

> 20% 是知识，80% 靠能力。

构建开放的知识体系，保持团队、个人快速成长，发挥整合的力量。

## 方式

基于相同领域内知识扩展学习。

## 来源

- 技术雷达：评估状态，已被公司某个项目使用；试验状态，说明在多个项目使用。总的来说，在技术雷达中的技术会比国内大公司的试用快半年左右。
- 公司邮件：项目的 Release，附带一些技术栈的使用。
- Github：通常会有一些趋势，供参考。

## 内容

### JavaScript 框架设计

锻炼：编程能力、架构能力、工程能力

### 在线广告 \ ADAS

锻炼：行业、业务知识的整合能力

### 项目管理

锻炼：提高组织效率的能力
