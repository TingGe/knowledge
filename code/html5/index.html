Html5特性检测

有四种基本技术可以用于检测浏览器是否支持某种HTML5特性，从简单到复杂依次是：

  • 检测全局对象(如window或navigator)是否拥有特定的属性。如检测地理位置特性
  • 创建一个元素，然后检测该元素的DOM对象是否拥有特定的属性。如检测画布特性
  • 创建一个元素，然后检测这个元素的DOM对象是否拥有特定的方法，同时调用这个方法并检查它的返回值。如检测支持的视频格式
  • 创建一个元素，给这个元素的DOM对象设定特定的属性值，然后检查浏览器是否保留了该属性值
